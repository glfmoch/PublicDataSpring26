I'm currently a Junior pursuing a Bachelor of Science in Data Visualization at UWB with an expected graduation date of June 2027.

My technical skills include R (tidyverse, ggplot2, Shiny, tmap), Tableau, basic SQL, basic Python (Pandas, Matplotlib), Excel, Java, and ArcGIS Pro. 

My project experience ranges from building an interactive Shiny dashboard using U.S. Census API data to explore cost-of-living trends across Washington State, to wrangling a 1.5 million-row tech job listing dataset to visualize labor market shifts in R. 

I have also applied these skills volunteering for Washington West African Center last summer as a Data Analyst Intern, analyzing community service data for a nonprofit and presenting Tableau visualizations to executive leadership to gain demographic insight and support funding efforts. 

Beyond the technical side, I am interested in how data visualization can help people understand complex data, bring issues to light, and positively impact communities. 

My interests lie in the gaming, automotive, and fashion industry which align with my hobbies, but I'm always open to exploring, whether it's venturing into non-profit, or learning more about environmental work.
